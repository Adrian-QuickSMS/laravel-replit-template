Build a Help Centre dashboard section for the QuickSMS portal using Bootstrap 5 and Fillow-style UI patterns.

The Help Centre module replaces the current “Support & Notifications” section and connects to HubSpot Service Hub via API.

The layout must contain three cards across the row:

1) Open Support Tickets
2) Knowledge Base Search
3) Platform Updates

Design requirements:

Use QuickSMS styling:

Primary colour: #7E57C2
Success: #2EAD4F
Warning: #F5A623
Muted text: #8B8FA3
Background: #F4F1FA
Card radius: 16px
Soft shadow cards
Bootstrap 5 spacing utilities
Bootstrap icons

Cards must match Fillow dashboard visual hierarchy.

--------------------------------------------------

CARD 1: Open Support Tickets (HubSpot Integration)

Title:

Open Support Tickets

Subtext:

Tickets that need your attention

Display dynamically from HubSpot:

Total open tickets
Awaiting reply count
In progress count
Resolved count

Example layout:

3 active tickets

• 1 awaiting reply
• 2 in progress
• 0 resolved

Buttons:

Primary:

View tickets

Secondary:

Create ticket

API:

Use HubSpot CRM Tickets API:

GET /crm/v3/objects/tickets

Filter tickets by:

pipeline stage != closed

If user email exists in session:

Filter tickets by contact association

Example:

associations.contact=email

--------------------------------------------------

CARD 2: Knowledge Base (HubSpot Knowledge Base Integration)

Title:

Knowledge Base

Subtext:

Search guides, API docs and setup help

Add:

Search input box

Placeholder:

Search help articles…

Below search:

Popular topics:

Getting started
API integration
RCS guides
Billing & payments
Account settings

Clicking search:

Calls HubSpot Knowledge Base Search API

GET /knowledge-base/v3/articles/search

Display dropdown results dynamically

Include:

Browse all articles button

Link:

/help-centre

CARD 3: Platform Updates

Title:

Platform Updates

Subtext:

Show:

System status indicator

Example:

All systems operational

1 new update

AUTHENTICATION

Use HubSpot Private App token:

.env variable:

HUBSPOT_PRIVATE_TOKEN

Requests:

Authorization: Bearer ${HUBSPOT_PRIVATE_TOKEN}

--------------------------------------------------

USER SESSION LINKING

If logged-in QuickSMS user email exists:

window.user.email

Use it to fetch associated tickets:

GET /crm/v4/objects/contacts/search

Then retrieve associated tickets
