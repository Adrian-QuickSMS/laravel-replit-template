3. STANDARDIZE ALL ADMIN PAGES:

Search for ALL files in /resources/views/admin/ that contain breadcrumbs or page headers and replace them with this standard structure:

FIND AND REPLACE PATTERN 1 (Custom admin-breadcrumb):

FIND:

<div class="admin-breadcrumb">
    <a href="{{ route('admin.dashboard') }}">Admin</a>
    <span class="separator">/</span>
    <span>PAGE_NAME</span>
</div>

REPLACE WITH:

<div class="row page-titles">
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a href="{{ route('admin.dashboard') }}">Admin</a></li>
        <li class="breadcrumb-item active">PAGE_NAME</li>
    </ol>
</div>

FIND AND REPLACE PATTERN 2 (Inconsistent page headers):

FIND variations like:

<h4 style="color: #1e3a5f; font-weight: 600;">Title</h4>
<h2 style="margin: 0; font-weight: 600; color: #1e3a5f;">Title</h2>
<h3>Title</h3>
REPLACE ALL WITH:

<div class="page-header">
    <div>
        <h2>PAGE_TITLE</h2>
        <p>Page description here</p>
    </div>
</div>
